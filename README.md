This repository contains code for a minimal computational study of bilateral sensing and symmetry-breaking processing in directional inference.

Stage 1 evaluates how bilateral sensor geometry shapes Z2-equivariant Fisher-information profiles across auditory, binocular, and tactile models.

Stage 2 evaluates how delay-based differential processing recovers directional sign under symmetric sensor placement using even and odd cross-correlation readout kernels.

Main claim:
Under bilaterally symmetric sensor geometry, body symmetry provides unbiased information uptake, whereas directional sign emerges only through symmetry-breaking processing at readout.
